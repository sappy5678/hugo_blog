最近在弄 spark 時遇到一堆破事，特此紀錄

collect 很久，甚至沒反應

conf = SparkConf().setMaster("local[*]") \
    .setAppName("App_Name") \
    .set('spark.executor.memory', '4G') \
    .set('spark.driver.memory', '45G') \
    .set('spark.driver.maxResultSize', '10G')
self.sc = SparkContext(conf=conf)
